Introducing cat-like typing detection

Have you ever had your cat type messages on your computer while you're away? To prevent confusion, we've created a new feature that detects and marks cat-like typing.

How does it work?

Using advanced pattern recognition and detection algorithms, cat-like typing is detected and flagged. Messages that contain cat-like typing have a notice to inform

If your messages are being flagged, it might be for one of the following reasons:

  • Keyword detection - your message contains specific words, phrases, or symbols that are associated with cat behavior.
  • Nearby key detection - your message contains letters, symbols or spaces from keys near to each other on a QWERTY keyboard.
